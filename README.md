# TAK Public Safety Community Policies
[TAK Public Safety](https://takps.org/about) (a 501(c)(3) non-profit) is proud to sponsor and maintain several community spaces as well as offer events to public safety professionals around the globe. These include:

* [The TAK End Users Slack](https://atakendusers.slack.com) \[[Policies](/slack)\]
* [TAKPS YouTube Channel](https://www.youtube.com/@takpublicsafety) \[[Policies](/youtube)\]
* [TAKPS Website](https://takps.org) \[[Policies](/website)\]
* [TAKPS Events](https://www.eventbrite.com/o/tak-public-safety-112496494451) \[[Policies](/eventbrite)\]
* [CivTAK Conference](https://www.civtakconf.com) \[[Policies](/civtakconf)\]
* [TAKPS LinkedIn](https://www.linkedin.com/company/takpublicsafety) \[[Policies](/linkedin)\]
* [TAKPS GitHub](https://github.com/TAK-Public-Safety) \[[Policies](/github)\]

To ensure clarity and understanding this repository holds the living documents for policies and procedures related to these sites. It is maintain by the TAK Public Safety organization, and any questions can be directed to contact at takps.org.
